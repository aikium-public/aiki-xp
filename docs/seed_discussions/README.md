# Seed GitHub Discussions

These are pre-written Q&A posts intended to be seeded into
[github.com/aikium-public/aiki-xp/discussions](https://github.com/aikium-public/aiki-xp/discussions)
shortly after launch. They anticipate the most common scientific questions
and let early visitors find answers without having to ask. The goal is to
signal that the project is actively maintained and that thoughtful answers
are expected.

## How to seed them

For each file in this directory (except this README):

1. Enable Discussions on the repo (Settings → Features → Discussions).
2. Pick the category that matches the filename prefix:
   - `ideas_*.md` → "Ideas"
   - `help_*.md` → "Q&A"
   - `show_*.md` → "Show and tell"
   - `general_*.md` → "General"
3. Create a new discussion, paste the file body, and post as the Aikium team.
4. Pin the three most useful ones so they stay visible at the top.
5. Mark your own answer as "accepted" where the file includes a proposed answer.

The individual files are written in the voice we want to project: direct,
quantitative, open about trade-offs.
